dlog Demo - Open your browser Console to view

This sample uses a config file to setup individual namespaces that should be shown, with optional style properties.

Module 1 & Module 2 log output would ordinarily be shown, as well as output from a direct call to dlog with a supplied namespace ('InlineModule') but in this sample, the showOnly option of dlog is used to suppress all other calls except the namespaces explicitly listed in the showOnly array.
